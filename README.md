# Virtual Mouse using Hand Gesture Recognition

## Overview
A contactless virtual mouse system that uses a webcam to detect 
hand gestures in real-time, allowing control of cursor movement, 
left click, right click, and volume control without any external hardware.

## Tech Stack
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## Features
- Real-time hand landmark detection via webcam
- Gesture-based cursor movement
- Left click, right click via finger gestures
- Volume control using hand gestures

## How to Run
1. Install dependencies: `pip install opencv-python mediapipe pyautogui`
2. Run: `python virtual_mouse.py`
